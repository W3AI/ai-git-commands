# bash git & devops shortcuts

## Mac install

copy files to /usr/local/bin
```bash
sudo cp * /usr/local/bin
```
might need to add permissions eg:
```bash
chmod 755 gc
```

## git

ga / git add .
gc / git commit -am 'comment'
gl / git log
gp / git push
gs / git status

## firebase

fd / firebase deploy

## npm, vue

nrs / npm run serve
nrb / npm run build
